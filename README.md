# Dark Mode with Reactjs, Gatsbyjs, Material-Ui(MUI) and Typescript

## How To Use
```terminal
$ git clone git@github.com:cieloazul310/gatsby-material-ui-darkmode.git
$ cd gatsby-material-ui-darkmode

# develop
$ npm run develop
or
$ yarn develop

# build
$ npm run build
or 
$ yarn build
```
